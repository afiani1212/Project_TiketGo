## Cara Menjalankan Project

### 1. Clone Repository
```bash
git clone https://github.com/afiani1212/Project_TiketGo.git
cd Ticket Booking
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Jalankan Server 

```bash
npm run dev
```

### 4. Akses Aplikasi

Buka browser dan akses:

```
http://localhost:5173
```
